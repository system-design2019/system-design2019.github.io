# T.I.M. 前端部署使用说明

本项目前端环境包括:

- Vue.js 2 
- vue-router 
- webpack2
- iView 3
- vuex
- animate
- axios
- vue-cookies

### 1. 项目本地运行【麻烦检查时请使用这种方式】

进入路径[system-design/frontend/TimeIsMoney]

- ！需要安装python

- ！当前目录下具有dist文件夹

执行

    python -m http.server 5678

在浏览器输入localhost:5678即可运行项目
